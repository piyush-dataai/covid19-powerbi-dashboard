# COVID-19 Analysis Dashboard (Power BI)

## Project Overview

This project presents an interactive COVID-19 Analysis Dashboard developed using Power BI. The dashboard provides insights into confirmed cases, recoveries, deaths, and active cases across multiple countries and regions.

## Objectives

* Analyze COVID-19 trends and patterns.
* Compare cases, recoveries, and deaths across countries.
* Visualize geographical distribution of cases.
* Track growth trends over time.

## Tools & Technologies

* Power BI
* Power Query
* DAX
* CSV Dataset

## Dashboard Features

* KPI Cards for Total Cases, Recoveries, Deaths, and Active Cases
* COVID Growth Trend Analysis
* Geographic Distribution Map
* Country-wise Comparison
* Recovery vs Active vs Death Analysis
* Interactive Filters (Country, Month, State)

## Key Insights

* Total COVID Cases: 11M
* Total Recovered: 9M
* Total Deaths: 261K
* Total Active Cases: 2M
